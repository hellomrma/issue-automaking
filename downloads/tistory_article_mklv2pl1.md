# 📱 아이폰 18 프로, 다이내믹 아일랜드가 더 작아진다고?

애플 팬들이 가장 궁금해하는 아이폰 18 프로의 변화 중 하나가 바로 다이내믹 아일랜드의 모습이에요. 최근 IT 업계 소식통들을 통해 흘러나오는 정보를 보면, 우리가 지금까지 봐왔던 알약 모양의 다이내믹 아일랜드가 상당히 달라질 것 같아요.

## 🔍 페이스ID가 화면 아래로 숨는다면?

가장 주목할 만한 변화는 페이스ID 관련 부품들이 디스플레이 아래쪽으로 이동한다는 소식이에요. 지금까지는 페이스ID를 위한 여러 센서들이 화면 위쪽에 모여있어서 다이내믹 아일랜드가 그 크기를 유지해왔는데, 이 부품들이 화면 뒤로 숨게 되면 자연스럽게 다이내믹 아일랜드도 작아질 수밖에 없겠죠.

IT 팁스터들의 말에 따르면, 앞으로는 전면 카메라와 적외선 카메라 정도만 화면에서 보이게 될 것 같아요. 그러니까 지금보다 훨씬 깔끔한 전면 디자인을 기대해볼 수 있을 것 같네요.

## 📍 카메라 위치도 바뀐다?

더 흥미로운 건 전면 카메라의 위치 변화예요. 일부 소식에 따르면 전면 카메라가 화면 왼쪽 상단으로 이동할 수도 있다고 해요. 만약 이게 현실이 된다면 우리가 알고 있는 가운데 위치의 다이내믹 아일랜드는 완전히 사라질 수도 있겠어요.

하지만 최근 공개된 렌더링 이미지들을 보면, 완전히 사라지기보다는 왼쪽 상단에 훨씬 작아진 형태로 남아있을 가능성이 높아 보여요. 어쨌든 지금보다는 화면을 훨씬 넓게 사용할 수 있게 될 것 같아서 기대가 되네요.

## 🔋 삼성 기술이 들어간다고?

국내 소식통에 따르면, 아이폰 18 프로에 삼성디스플레이의 언더스크린 적외선 기술이 적용될 수 있다고 해요. 이 기술이 정말 탑재된다면 페이스ID가 화면 아래에서 작동하는 게 훨씬 수월해질 거예요.

게다가 LTPO+ 디스플레이 기술도 함께 적용돼서 배터리 효율까지 개선될 것 같아요. 화면은 더 넓어지고, 배터리는 더 오래가고, 정말 일석이조가 아닐까요?

## 💭 사용자 경험은 어떻게 달라질까?

다이내믹 아일랜드가 작아지거나 위치가 바뀐다면, 우리가 스마트폰을 사용하는 방식도 조금씩 달라질 것 같아요. 지금까지는 다이내믹 아일랜드를 중심으로 알림이나 음악 재생 정보가 표시됐는데, 이런 UI도 함께 변화할 가능성이 높아요.

무엇보다 화면을 더 넓게 사용할 수 있게 되면, 동영상 시청이나 게임할 때 몰입감이 훨씬 좋아질 것 같아요. 특히 세로로 긴 콘텐츠를 볼 때 방해받는 요소가 줄어드니까 더 쾌적한 경험을 할 수 있을 거예요.

물론 아직 확정된 건 아니지만, 애플이 계속해서 화면 공간 활용도를 높이려고 노력하고 있는 걸 보면 이런 변화들이 현실이 될 가능성이 꽤 높아 보여요. 올 가을 아이폰 18 프로가 공개되면 정말 어떤 모습일지 궁금하네요.
